money to chinese
===

JavaScript function for converting amount to Chinese uppercase.

## Installation

```bash
npm install moneyToChinese
```

## Usage

```js
import moneyToChinese form 'moneyToChinese';

moneyToChinese(100.15);
//⇒ "壹佰元壹角伍分"

moneyToChinese('100.15');
//⇒ "壹佰元壹角伍分"
```
